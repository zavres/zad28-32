1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

Sposób 1:

a) nazwa_obiektu.nazwa_właściwości = "tekst"; np. auto.marka = "Opel Omega";

b) nazwa_obiektu[nazwa_właściwości]; np. auto['cena'] = 25000;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Do właściwości i metod można się odwołać, umieszczając nazwę obiektu przed ich nazwą, i dodając kropkę np. console.log(samochod1.kolor);

3. Wymień cechy charakterystyczne konstruktora.

  • Jest uruchamiany automatycznie podczas tworzenia obiektu danej klasy.
  • Ma taką samą nazwę jak klasa.
  • Nie zwraca żadnej wartości.
  • Może przyjmować argumenty, podobnie jak zwykła funkcja czy metoda, aby tworzyć obiekt za pomocą operatora new.

4. Konstruowanie obiektu na podstawie przykładu.

    var osoba_1 = {
      nazwisko: 'Kowalski',
      imie: 'Jan',
      zawod: 'piekarz',
      wyświetl: function() {
        document.write(this.nazwisko + ' ' + this.imie);
      }
    };

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Słowo kluczowe this pozwala odwołać się do pola lub metody obiektu z wnętrza tego obiektu. Pola i metody muszą być oddzielone przecinkami w obrębie jednego obiektu.

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

    function klient(nazwisko_k, imie_k, zawod_k) {
      this.nazwisko = nazwisko_k;
      this.imie = imie_k;
      this.zawod = zawod_k;
      this.wypisz = function() {
        alert(this.nazwisko + ' ' + this.imie);
      };
    }

7. Znaczenie słowa kluczowego new wraz z przykładem.

    var osoba1 = new klient('Kowalski', 'Jan', 'kierowca');
    var osoba2 = new klient('Nowak', 'Anna', 'sekretarka');

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem (tekst pogrubiony).

Właściwość prototype jest wykorzystywana do dodawania metod i właściwości do obiektu. Daje możliwość rozszerzenia konstruktora, dodając nowe metody lub właściwości do wszystkich obiektów utworzonych na podstawie tego konstruktora.

    function klient() {
      this.nazwisko = 'Olszewski';
      this.imie = 'Gracjan';
    }

    klient.prototype.pisz_dane = function() {
      document.write(this.nazwisko + ' ' + this.imie);
    };

    klient.prototype.zawod = 'kierowca';

    var osoba1 = new klient();
    osoba1.pisz_dane();
